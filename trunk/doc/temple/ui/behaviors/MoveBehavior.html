Classpublic class MoveBehavior
InheritanceMoveBehavior BoundsBehavior AbstractDisplayObjectBehavior AbstractBehavior CoreEventDispatcher flash.events.EventDispatcher
Implements IEnableable

The MoveBehavior makes a DisplayObject movable by the cursor keys. The MoveBehavior uses the decorator pattern, so you won't have to change the code of the DisplayObject. When the DispayObject has focus, use the cursor keys to move the object one pixel. If you also press the SHIFT key, the object will move 10 pixels.

It is not nessessary to store a reference to the MoveBehavior since the MoveBehavior is automatically destructed when the DisplayObject is destructed.

If you want the object to be available for garbage collection make sure you:
  • Remove all event listeners on this object (use removeAllEventListeners on Temple objects).
  • Remove all event listeners from this object.
  • Set all non-primitive variables to null.
  • Set all references to this object to null in other objects.

Examples

If you have a MovieClip called 'mcClip' add MoveBehavior like:

     new MoveBehavior(mcClip);

If you want to limit the moving to a specific bounds, you can add a Rectangle. By adding the Reactangle you won't be able to move DisplayObject outside the Rectangle:

     new MoveBehavior(mcClip, new Rectangle(100, 100, 200, 200);

It's also possible to define a child object as a 'MoveButton'. A MoveButton is a DisplayObject that is used to move the target.

If you have a MovieClip called 'mcClip' with a child called 'mcMoveButton' add MoveBehavior like:

     new MoveBehavior(mcClip, null, mcClip.mcMoveButton);
