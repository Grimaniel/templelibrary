Classpublic class ButtonBehavior
InheritanceButtonBehavior AbstractButtonBehavior AbstractDisplayObjectBehavior AbstractBehavior CoreEventDispatcher flash.events.EventDispatcher

The ButtonBehavior can make every DisplayObject act like a button. The ButtonBehavior keeps track of status of the button. The status of the button indicates if the mouse is currently up, over or down the button or if the button is selected of disabled. The the reaction of mouseOver and mouseOut Events can be delayed.

The status of the buttons is dispatch in a tunneled ButtonEvent on all the children (and grandchildren etc.) of the button. This Button is used by ButtonDesignBehaviors to show the current status.

destruct()method
override public function destruct():void

Destructs the object and makes it available for garbage collection.

When overriding this method, always call super.destruct() at the end!

If you want the object to be available for garbage collection make sure you:
  • Remove all event listeners on this object (use removeAllEventListeners on Temple objects).
  • Remove all event listeners from this object.
  • Set all non-primitive variables to null.
  • Set all references to this object to null in other objects.
When a Temple object is destructed a DestructEvent.DESTRUCT is dispatched from the object (if the object implements IDestructibleEventDispatcher).

Note: Bear in mind that it is possible an object can be destructed more than once.
