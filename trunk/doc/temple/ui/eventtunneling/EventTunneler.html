InheritanceEventTunneler AbstractDisplayObjectBehavior AbstractBehavior CoreEventDispatcher flash.events.EventDispatcher
Implements IDebuggable

An EventTunneler will automaticly dispatch TunnelEvents on all children (and grant-children) of a Sprite if there is a TunnelEvent dispatched on the Sprite.
