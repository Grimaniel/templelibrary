ISubmitButton
 
Packagetemple.ui.form.components
Interfacepublic interface ISubmitButton extends IDisplayObjectContainer , IDisplayObject , IEventDispatcher , IBitmapDrawable
Implementors SubmitButton

A button which is automatically used for submitting a form.

This is an empty interface which is only used to identify the button. Use this interface for a button in combination with a FormCompontent. When you set this button on the display list of a FormCompontent, this button will automatically be set as SubmitButton for the FormCompontent.

The Temple knows different kinds of buttons. Check out the button schema in the UI Module of the Temple for a list of all available buttons which their features.
