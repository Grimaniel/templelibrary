Packagetemple.core.display
Interfacepublic interface IDisplayObjectContainer extends IDisplayObject , IEventDispatcher , IBitmapDrawable

Interface that contains all the properties of a DisplayObjectContainer. This interface is implemented by all DisplayObjectContainesr of the Temple, but not by Flash native DisplayObjectContainers. This interface is extended by other interface to force they can only be implemented by DisplayObjectContainers.
