Common Module
 

About

This is version 3.3.0 of the Temple Common Module, built on 2012-12-13. Get the latest version from: http://code.google.com/p/templelibrary/. You can get the files from SVN or you could download it as a zip from the downloads page.

This module contains some general classes (enumerators, events and interfaces) which are used in other Temple modules. This module does not contain any logic, but is only needed for other modules.

Content

This module contains the following folders:

/bin
Contains the SWC of this module.
/doc
Contains the documentation of the common module.
/examples
Contains some examples of the common module.
/lib
Contains the SWC files of other Temple modules which are needed for this module.
/source
Contains the source (.as) files of this module.
/tools
Contains ANT scripts which are used to generate docs, examples, swc etc...
(This folder is not available in the zip file.)

Dependencies

This module is only dependent on the core module. But almost all other Temple modules (except for the core) are dependent on this module. An SWC of the core can be found in the /lib folder.

Getting started

Adding the common module to your project

To add the common module to you project you could use the SWC file or the AS files. The SWC file can be found in the /bin folder. If you want to use the SWC file you need to add the SWC file to the class path of your project. You also need to add to SWC of the core to the class path of your project. This SWC can be found in the /lib folder.

If you want to use the AS files you just need to copy the content of the /source folder to the source folder of your project. Or you could add the /source folder to the class patch of your project.
NOTE: don't copy folders from an SVN check out, since this can cause SVN problems. It would be better do an SVN export to your own folder.

Examples

Check out the /examples folder for all examples. This folder also contains a list of all available examples.
