Net Module
 

About

This is version 3.5.0 of the Temple Net Module, built on 2013-01-30. Get the latest version from: http://code.google.com/p/templelibrary/. You can get the files from SVN or you could download it as a zip from the downloads page.

This module contains classes for sending and receiving data over the net.

Content

This module contains the following folders:

/bin
Contains the SWC of this module.
/doc
Contains the documentation of the Net module.
/lib
Contains the SWC files of other Temple modules which are needed for this module.
/source
Contains the source (.as) files of this module.
/tools
Contains ANT scripts which are used to generate docs, examples, swc etc...
(This folder is not available in the zip file.)

Dependencies

This module is only dependent on the core module. The SWC of the core module can be found in the /lib folder.

Getting started

Adding the Net module to your project

To add the Net module to you project you could use the SWC file or the AS files. The SWC file can be found in the /bin folder. If you want to use the SWC file you need to add the SWC file to the class path of your project. You also need to add the dependent SWC's to the class path of your project. These SWCs can be found in the /lib folder.

If you want to use the AS files you just need to copy the content of the /source folder to the source folder of your project. Or you could add the /source folder to the class patch of your project.
NOTE: Don't copy folders from an SVN check out, since this can cause SVN problems. It would be better to do an SVN export to your own folder.
