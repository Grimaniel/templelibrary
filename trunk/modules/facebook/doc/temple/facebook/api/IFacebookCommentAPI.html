getComment()method 
public function getComment(commentId:String, callback:Function = null, fields:FacebookCommentFields = null, params:Object = null, forceReload:Boolean = false):IFacebookCall

Get a single comment. If successful, the result contains an IFacebookCommentData object.

Parameters

commentId:String — the id of the comment.
 
callback:Function (default = null) — a callback method which must be called when the data is ready. This callback must accept one (and only one) argument of type IFacebookResult. If the call was successful the success Boolean of the result is true and the photos are in the data property of the result object.
 
fields:FacebookCommentFields (default = null) — a FacebookCommentFields object with all the requested fields set to true.
 
params:Object (default = null) — option params to send with the request.
 
forceReload:Boolean (default = false) — when caching is enabled you can force the service to reload the data and not get the cached data when setting this value to true.

Returns
IFacebookCall

public function getComments(objectId:String, callback:Function = null, offset:Number, limit:Number, fields:FacebookCommentFields = null, params:Object = null, forceReload:Boolean = false):IFacebookCall

Get all the comments on an object. If successful, the result contains an Array with IFacebookCommentData objects.

Parameters

objectId:String — the id of the object with the comments
 
callback:Function (default = null) — a callback method which must be called when the data is ready. This callback must accept one (and only one) argument of type IFacebookResult. If the call was successful the success Boolean of the result is true and the photos are in the data property of the result object.
 
offset:Number (default = NaN) — the position of the first item in the list
 
limit:Number (default = NaN) — the maximum amount of items.
 
fields:FacebookCommentFields (default = null) — a FacebookCommentFields object with all the requested fields set to true.
 
params:Object (default = null) — option params to send with the request.
 
forceReload:Boolean (default = false) — when caching is enabled you can force the service to reload the data and not get the cached data when setting this value to true.

Returns
IFacebookCall
