Packagetemple.ui.buttons.behaviors
Interfacepublic interface IButtonDesignBehavior extends IBehavior , INestableButton , ICoreEventDispatcher , IEventDispatcher , IDestructible

IButtonDesignBehavior is an interface for ButtonDesignBehaviors. ButtonDesignBehaviors listens for ButtonEvents on a DisplayObject and changes the design if the DisplayObject corresponding to the status of the ButtonEvent.
