getProperties()method 
public static function getProperties(object:Object, accessorAccess:String = null, namespaces:Vector.<Namespace> = null):Vector.<String>

Get the keys and properties of an object.

Parameters

object:Object — the object to get the properties of.
 
accessorAccess:String (default = null) — the type of access (read, write, readwrite) of the properties. If null, all properties are returned
 
namespaces:Vector.<Namespace> (default = null) — a list of namespaces of the properties you want to receive. If null, only properties without namespace are returned. If the list is empty (length is 0), properties of all namespaces are returned

Returns
Vector.<String> — an Array of all the keys
