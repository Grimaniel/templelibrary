Packagetemple.ui.states.error
Interfacepublic interface IErrorState extends IState , IEnableable , IDisplayObject , IShowable , IEventDispatcher , IBitmapDrawable
Implementors ErrorFadeState

A DisplayObject that implements IErrorState represents the error state of his parent.

The IErrorState is used by FormElements for displaying errors. If there is an error message attached to the error, the IErrorState will get the message through the 'message' setter, so the IErrorState can display the error message.
