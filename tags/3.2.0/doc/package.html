dump()function
public function dump(object:Object, depth:uint = 3, duplicates:Boolean = true, constants:Boolean = false, namespaces:Boolean = false, methods:Boolean = false, dates:Boolean = false):String

Recursively dumps the object and all his properties.

Parameters

object:Object — object to dump
 
depth:uint (default = 3) — indicates the recursive factor
 
duplicates:Boolean (default = true) — indicates if duplicate objects should be dumped again
 
constants:Boolean (default = false) — indicates if constants will be included (true) or not (false)
 
namespaces:Boolean (default = false) — indicates if methods will be included (true) or not (false)
 
methods:Boolean (default = false) — indicates if a Date will be fully dumped (including all properties) (true) or only as a simple String (false)
 
dates:Boolean (default = false)

Returns
String — the object tree as String

Example 
     
      trace(dump(myObject));
log()function 
public function log(message:*, object:* = __UNLOGGABLE_STRING__, depth:uint = 0, duplicates:Boolean = false, level:String = info):void

Quick way to send a message to the Log class.

Parameters

message:* — the message which is sent to the Log.
 
object:* (default = __UNLOGGABLE_STRING__) — an object wich is added to the message. The object will be added with all his properties, using the dump() method.
 
depth:uint (default = 0) — indicates the depth of recursion of the properties of the object. If set to 0 only the properties of the object will be added, if set to 1 also the properties of each property will be added etcetera.
 
duplicates:Boolean (default = false) — a Boolean which indicates if duplicate objects in the properties of the object should be added every time the object occurs (true) or only should be added once and ignored for the other occasions.
 
level:String (default = info) — the level of the log message. Possible values are "error", "warn", "debug", "info", "fatal" and "status".

See also

temple.core.debug.log.LogLevel

Example 
     
      log("myObject", myObject);
