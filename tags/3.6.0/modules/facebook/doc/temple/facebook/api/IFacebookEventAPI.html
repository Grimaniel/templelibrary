getEvent()method 
public function getEvent(eventId:String, callback:Function = null, fields:FacebookEventFields = null, params:Object = null, forceReload:Boolean = false):IFacebookCall

Get an event. If successful, the result contains an IFacebookEventData object.

Parameters

eventId:String — the id of the event.
 
callback:Function (default = null) — a callback method which must be called when the data is ready. This callback must accept one (and only one) argument of type IFacebookResult. If the call was successful the success Boolean of the result is true and the photos are in the data property of the result object.
 
fields:FacebookEventFields (default = null) — a FacebookEventFields object with all the requested fields set to true.
 
params:Object (default = null) — option params to send with the request.
 
forceReload:Boolean (default = false) — when caching is enabled you can force the service to reload the data and not get the cached data when setting this value to true.

Returns

public function getEvents(callback:Function = null, userId:String = me, offset:Number, limit:Number, fields:FacebookEventFields = null, params:Object = null, forceReload:Boolean = false):IFacebookCall

Get all the events of a user. If successful, the result contains an Array with IFacebookEventData objects.

Parameters

offset:Number (default = NaN) — the position of the first item in the list
 
limit:Number (default = NaN) — the maximum amount of items.
 
fields:FacebookEventFields (default = null) — a FacebookEventFields object with all the requested fields set to true.
 
params:Object (default = null) — option params to send with the request.
