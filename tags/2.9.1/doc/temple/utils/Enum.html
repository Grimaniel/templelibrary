Classpublic final class Enum
InheritanceEnum Object

Reads the values of all constants (optionally of specified type) to an Array or Object. var myConstantValues:Array = Enum.getEnumArray(MyVarsClass, Enum.STRING); or var myConstantNameValues:Object = Enum.getEnumHash(MyVarsClass, Enum.STRING);
