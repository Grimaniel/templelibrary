public function StageSettings(displayObject:DisplayObject, stageAlign:String = TL, stageScaleMode:String = noScale, width:Number, height:Number)

Create a new StageSettings object

Parameters
displayObject:DisplayObject — a display object which is used to get the stage.
 
stageAlign:String (default = TL) — alignment of the stage.
 
stageScaleMode:String (default = noScale) — scalemode of the stage.
 
width:Number (default = NaN) — width of the stage. This value will be available in StageSettings.width and can be used trough the application to get to original stage width.
 
height:Number (default = NaN) — height of the stage. This value will be available in StageSettings.height and can be used trough the application to get to original stage height.

destruct()method
override public function destruct():void

Destructs the object and makes it available for garbage collection.

When overriding this method, always call super.destruct() at the end!

If you want the object to be available for garbage collection make sure you:
  • Remove all event listeners on this object (use removeAllEventListeners on Temple objects).
  • Remove all event listeners from this object.
  • Set all non-primitive variables to null.
  • Set all references to this object to null in other objects.
When a Temple object is destructed a DestructEvent.DESTRUCT is dispatched from the object (if the object implements ICoreEventDispatcher).

Note: Bear in mind that it is possible an object can be destructed more than once.
