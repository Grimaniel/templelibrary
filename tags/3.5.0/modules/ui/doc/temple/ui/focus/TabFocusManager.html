Classpublic class TabFocusManager
InheritanceTabFocusManager CoreEventDispatcher flash.events.EventDispatcher
Implements IFocusable, IDebuggable, IEnableable

Class for managing focus on UI components that implement IFocusable.

Use of this class overrides default tab button behavior. Only UI components added to an instance of this class can get focus through use of the tab key. Consecutive presses on the TAB key will cycle through the list of items in order of addition, unless a specific tab index has been set on any of them. With SHIFT-TAB the list is cycled in reversed order.

override public function destruct():void

Destructs the object and makes it available for garbage collection.

When overriding this method, always call super.destruct() at the end!

If you want the object to be available for garbage collection make sure you:
  • Remove all event listeners on this object (use removeAllEventListeners on Temple objects).
  • Remove all event listeners from this object.
  • Set all non-primitive variables to null.
  • Set all references to this object to null in other objects.
When a Temple object is destructed a DestructEvent.DESTRUCT is dispatched from the object (if the object implements ICoreEventDispatcher).

Note: Bear in mind that it is possible an object can be destructed more than once.

Examples
Suppose two objects of type InputField have been defined on the timeline, with instance names "mcName" & "mcEmail". Use the following code to allow focus management on them: 
         focusManager = new TabFocusManager();
         focusManager.add(mcName);
         focusManager.add(mcEmail);
