Classpublic class StylableLabelButton
InheritanceStylableLabelButton LabelButton Label CoreMovieClip flash.display.MovieClip
Implements IStylableLabel

A StylableLabelButton is a LabelButton which can be styled using CSS.

The Temple knows different kinds of buttons. Check out the button schema in the UI Module of the Temple for a list of all available buttons which their features.
