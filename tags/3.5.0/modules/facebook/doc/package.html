public function dump(object:Object, depth:uint = 3, duplicates:Boolean = true, constants:Boolean = false, namespaces:Boolean = false, methods:Boolean = false, dates:Boolean = false, skipEmpty:Boolean = false):String

Recursively dumps the object and all his properties.

Parameters

object:Object — object to dump
 
depth:uint (default = 3) — indicates the recursive factor
 
duplicates:Boolean (default = true) — indicates if duplicate objects should be dumped again
 
constants:Boolean (default = false) — indicates if constants will be included (true) or not (false)
 
namespaces:Boolean (default = false) — indicates if methods will be included (true) or not (false)
 
methods:Boolean (default = false) — indicates if a Date will be fully dumped (including all properties) (true) or only as a simple String (false)
 
dates:Boolean (default = false) — is set to true properties with a value of null or NaN will be skipped
 
skipEmpty:Boolean (default = false)

Returns
String — the object tree as String

Example 
     
      trace(dump(myObject));
