public function getCheckin(checkinId:String, callback:Function = null, fields:FacebookCheckinFields = null, params:Object = null, forceReload:Boolean = false):IFacebookCall

Get a checkin. If successful, the result contains an IFacebookCheckinData object.

Parameters

checkinId:String — the id of the checkin.
 
callback:Function (default = null) — a callback method which must be called when the data is ready. This callback must accept one (and only one) argument of type IFacebookResult.
