Interfacepublic interface IFormService extends ICoreEventDispatcher , IEventDispatcher , IDestructible
Implementors MultiFormService, ObjectFormService, XMLFormService

A IFormService handles the data of a Form. When a Form is submitted it send his data to a IFormService, using the 'submit' method. After the IFormService handled the data is returns an IFormResult (only if the data is handled synchronously, otherwise null is returned) and dispatched an FormServiceEvent.

submit()method
public function submit(data:Object):IFormResult

Handles the form data. If the data is handled synchronously an IFormResult is returned. If the data is handled asynchronously, null is returned. After submit is complete a FormServiceEvent is dispatched.
