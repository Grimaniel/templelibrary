Utils Module
 

About

This is version 3.2.0 of the Temple Utils Module, built on 2012-10-11. Get the latest version from: http://code.google.com/p/templelibrary/. You can get the files from SVN or you could download it as a zip from the downloads page.

This module contains some handy utility classes which can be used out of the box and are used in other Temple modules. Static utility methods are grouped together in a single file based on the type of object they apply to. All these files with static utility method for a single object are grouped together in the sub package 'temple.utils.types'. So all utility methods for Arrays can be found in the ArrayUtils class, utility methods for DisplayObjects can be found in the DisplayObjectUtils class, etc.

Content

This module contains the following folders:

/bin
Contains the SWC of this module.
/doc
Contains the documentation of the utils module.
/examples
Contains some examples of the utils module.
/lib
Contains the SWC files of other Temple modules which are needed for this module.
/source
Contains the source (.as) files of this module.
/extended
Contains the source (.as) files of some additional classes of this module. These additional class have more dependencies to other modules which are not supplied with this module.
/tools
Contains ANT scripts which are used to generate docs, examples, swc etc...
(This folder is not available in the zip file.)

Dependencies

This module is dependent on the core module and the common module. The SWCs of the core and common modules can be found in the /lib folder.

Classes inside the /extended folder do have more dependencies to other Temple modules. The SWC files of these modules are not supplied with this module. If you want to use these additional classes you will need these other modules. The modules you need varies per class. Check the imports of the class to define the modules you need.

Getting started

Adding the common module to your project

To add the common module to you project you could use the SWC file or the AS files. The SWC file can be found in the /bin folder. If you want to use the SWC file you need to add the SWC file to the class path of your project. You also need to add the SWC of the core and common to the class path of your project. These SWCs can be found in the /lib folder.

If you want to use the AS files you just need to copy the content of the /source folder to the source folder of your project. Or you could add the /source folder to the class patch of your project.
NOTE: Don't copy folders from an SVN check out, since this can cause SVN problems. It would be better to do an SVN export to your own folder.

Examples

Check out the /examples folder for all examples. This folder also contains a list of all available examples.
