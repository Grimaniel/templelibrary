Classpublic final class CodeComponents
InheritanceCodeComponents Object

CodeComponents are code-only, easy-to-use, light-weight components for Flash. CodeComponents are special designed for quick and easy creation of user interfaces and forms. All CodeComponents extend Temple classes and there they have all there features.

CodeComponents can be styled by changing the properties in the CodeStyle class. This must be done before the component is created. Components which are already created before the properties of the CodeStyle class are changed won't be affected.
