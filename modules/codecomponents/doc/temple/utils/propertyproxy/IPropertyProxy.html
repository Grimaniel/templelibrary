Packagetemple.utils.propertyproxy
Interfacepublic interface IPropertyProxy extends IDestructible , ICancellable

An IPropertyProxy set a property on an Object to a given value. The IPropertyProxy can manipulate the value before it's set. Therefor you can e.g. add a Tween to a property or round the value.
