TweenLite Module
 

About

This is version 3.0.1 of the Temple TweenLite Module, built on 2012-03-01. Get the latest version from: http://code.google.com/p/templelibrary/.

TweenLite is create by GreenSock and has it's own license. We made some minor changes to make it more collaborative with the Temple.

For more information about TweenLite, visit: www.greensock.com/tweenlite/.

Content

This module contains the following folders:

/bin
Contains the SWC of this module.
/doc
Links to the online documentation of TweenLite.
/lib
Contains the SWC files of other Temple modules which are needed for this module.
/source
Contains the source (.as) files of this module.
/tools
Contains ANT scripts which are used to generate docs, examples, swc etc...
(This folder is not available in the zip file.)

Dependencies

This version of TweenLite is only dependent on the core module of the Temple.
