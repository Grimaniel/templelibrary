create()method
public function create(message:String, id:String = me, callback:Function = null, picture:String = null, link:String = null, name:String = null, caption:String = null, description:String = null, source:String = null, objectAttachment:String = null):IFacebookCall

Publish a new post on the given profile's feed/wall

February 2013 Breaking Changes: Removing ability to post to friends walls via Graph API.

Parameters

message:String — The message
 
id:String (default = me) — The id of the object where this post belangs to
 
callback:Function (default = null) — A callback method which must be called when this call is finished. This callback must accept one (and only one) argument of type IFacebookResult. If the call was successful the success Boolean of the result is true and the photos are in the data property of the result object.
 
picture:String (default = null) — A link to the picture included with this post
 
link:String (default = null) — The link attached to this post
 
name:String (default = null) — The name of the link
 
caption:String (default = null) — The caption of the link (appears beneath the link name)
 
description:String (default = null) — A description of the link (appears beneath the link caption)
 
source:String (default = null) — A URL to a Flash movie or video file to be embedded within the post
 
objectAttachment:String (default = null) — A PhotoID from an uploaded photo to a facebook album. Use this instead of an photo-url outside facebook when you want to share photo that you've uploaded to facebook

Returns
IFacebookCall

public function getPost(id:String, callback:Function = null, fields:FacebookPostFields = null, params:Object = null, forceReload:Boolean = false):IFacebookCall

Get a single post. If successful, the result contains a IFacebookPostData object.

Parameters

id:String — the id of the post.
 
callback:Function (default = null) — a callback method which must be called when the data is ready. This callback must accept one (and only one) argument of type IFacebookResult. If the call was successful the success Boolean of the result is true and the photos are in the data property of the result object.
 
fields:FacebookPostFields (default = null) — a FacebookPostFields object with all the requested fields set to true.
 
params:Object (default = null) — option params to send with the request.
 
forceReload:Boolean (default = false) — when caching is enabled you can force the service to reload the data and not get the cached data when setting this value to true.

Returns

getPosts()method 
public function getPosts(callback:Function = null, id:String = me, since:Date = null, until:Date = null, fields:FacebookPostFields = null, params:Object = null, forceReload:Boolean = false):IFacebookCall

Get all the post on the wall of a user or event.

If successful, the result contains an Array with IFacebookPostData objects.

Note: February 2013 Breaking Changes: Offset no longer allowed when searching posts.

Parameters

callback:Function (default = null) — a callback method which must be called when the data is ready. This callback must accept one (and only one) argument of type IFacebookResult. If the call was successful the success Boolean of the result is true and the photos are in the data property of the result object.
 
id:String (default = me) — the id of the user (or the album for photos or event for invites).
 
since:Date (default = null) — the position of the first item in the list
 
until:Date (default = null) — the maximum amount of items.
 
fields:FacebookPostFields (default = null) — a FacebookPostFields object with all the requested fields set to true.
 
params:Object (default = null) — option params to send with the request.
 
forceReload:Boolean (default = false) — when caching is enabled you can force the service to reload the data and not get the cached data when setting this value to true.

Returns
IFacebookCall
