get()method 
public function get(callback:Function = null, method:String = null, id:String = me, objectClass:Class = null, params:Object = null, fields:IFacebookFields = null, forceReload:Boolean = false):IFacebookCall

Get data from Facebook.

Parameters

callback:Function (default = null) — a function that must be called when the data is retreived from Facebook and parsed. Note: this function must accept one, and only one, argument of type IFacebookResult.
 
method:String (default = null) — the Facebook method to call
 
id:String (default = me) — and optional ID of the object which must be get
 
objectClass:Class (default = null) — an optional class which is used for parsing the data to. This class must implement IObjectParsable. This class can also be provided using the registerVO method.
 
params:Object (default = null) — an optional object for extra parameters
 
fields:IFacebookFields (default = null) — an object which defines which fields must be returned. This fields object is also used for automatically permissions retrieving.
 
forceReload:Boolean (default = false) — if set to true, the service will always reload the requested data, even when cache is enabled.

Returns
