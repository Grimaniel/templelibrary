Data Module
 

About

This is version 3.3.0 of the Temple Data Module, built on 2012-12-13. Get the latest version from: http://code.google.com/p/templelibrary/. You can get the files from SVN or you could download it as a zip from the downloads page.

This module contains classes for loading, parsing, caching, encoding, storing and handling all kinds of data, such as XML, JSON or images. Check out the documentation for all classes of the the data module.

Content

This module contains the following folders:

/bin
Contains the SWC of this module.
/doc
Contains the documentation of the data module.
/examples
Contains some examples of the data module.
/lib
Contains the SWC files of other Temple modules which are needed for this module.
/source
Contains the source (.as) files of this module.
/tools
Contains ANT scripts which are used to generate docs, examples, swc etc...
(This folder is not available in the zip file.)

Dependencies

This module is dependent on the core, common and utils modules. The SWCs of these modules can be found in the /lib folder.

Getting started

Adding the common module to your project

To add the common module to you project you could use the SWC file or the AS files. The SWC file can be found in the /bin folder. If you want to use the SWC file you need to add the SWC file to the class path of your project. You also need to add to SWC of the core, common and utils to the class path of your project. These SWCs can be found in the /lib folder.

If you want to use the AS files you just need to copy the content of the /source folder to the source folder of your project. Or you could add the /source folder to the class patch of your project.
NOTE: don't copy folders from an SVN check out, since this can cause SVN problems. It would be better to do an SVN export to your own folder.

Examples

Check out the /examples folder for all examples. This folder also contains a list of all available examples.

Note: every example has an XML file which is an ANT build file which can be used to build the example.
