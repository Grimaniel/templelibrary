This class uses a timeline animation to display the state object.

The timeline can be used using framelabels or without framelabels

With framelabel Use 'show' as label to define the show animation. Use 'hide' to define the hide animation. The animation will automatically stop on the last frame of the animation. DO NOT use frame scripts to stop the animation!!!

Withoutframelabel The first frame of the animation is use as hide, the last frame for show. The timeline animation will be played forward and backwards to toggle between these states.
