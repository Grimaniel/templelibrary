Interfacepublic interface IOptionalValidationRule extends IValidationRule , IValidatable , IDestructible
Implementors MinimumLengthValidationRule, RegExpValidationRule

Interface for validation rules which can be optional. Optional means that an empty value is also valid. For instance when you set the optional property of an EmailValidationRule to true, the user can leave the email input blank. If the user fills in something in the email input, it must be an email.
