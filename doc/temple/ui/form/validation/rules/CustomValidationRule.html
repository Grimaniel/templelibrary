public function CustomValidationRule(target:IHasValue, rule:Function, params:Array = null)

Create your own custom validation. This validation rule can not be added to the form bij using the 'addFormElement' method, since the 2nd argument is mandetory For adding a CustomValidationRule to the form, use form.validator.addValidationRule

Parameters
target:IHasValue — The form element with needs to be validated (ig an InputField), target must implement IValidatable
 
rule:Function — The funtion that need be called on validation. The function needs to return a Boolean, where true means valid and false not valid.
 
params:Array (default = null) — (optional): the arguments for the rule that are applied on the function when called

destruct()method
override public function destruct():void

Destructs the object and makes it available for garbage collection.

When overriding this method, always call super.destruct() at the end!

If you want the object to be available for garbage collection make sure you:
  • Remove all event listeners on this object (use removeAllEventListeners on Temple objects).
  • Remove all event listeners from this object.
  • Set all non-primitive variables to null.
  • Set all references to this object to null in other objects.
When a Temple object is destructed a DestructEvent.DESTRUCT is dispatched from the object (if the object implements ICoreEventDispatcher).

Note: Bear in mind that it is possible an object can be destructed more than once.
