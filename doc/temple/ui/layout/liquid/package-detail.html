temple.ui.layout.liquid Interfaces  | Classes
 


Interfaces
 InterfaceDescription
 ILiquidObject A liquid object is an object witch positon (x and y) and dimension (width and height) is influenced by his 'parent' (related object) Instead of just setting an objects x and y, you can set the objects left, right, top and/or bottom.
 ILiquidRelatedObject An ILiquidRelatedObject can be used by an ILiquidObject as related object.
Classes
 ClassDescription
 LiquidBehavior Class add liquid behavior on a DisplayObject.
 LiquidContainer This class is used as a wrapper around other liquid objects to set special liquid properties.
 LiquidMovieClip The LiquidMovieClip extends the CoreMovieClip but has already LiquidBehavior implemented.
 LiquidSprite The LiquidSprite extends the CoreMovieClip but has already LiquidBehavior implemented.
 LiquidStage Singleton class for wrapping the Stage as a LiquidObject for aligning other LiquidObject to the stage.
