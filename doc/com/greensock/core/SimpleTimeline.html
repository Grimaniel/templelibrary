Classpublic class SimpleTimeline
InheritanceSimpleTimeline TweenCore Object

SimpleTimeline is the base class for the TimelineLite and TimelineMax classes. It provides the most basic timeline functionality and is used for the root timelines in TweenLite. It is meant to be very fast and lightweight. Copyright 2010, GreenSock. All rights reserved. Property Detail
autoRemoveChildrenproperty
public var autoRemoveChildren:Boolean

If a timeline's autoRemoveChildren is true, its children will be removed and made eligible for garbage collection as soon as they complete. This is the default behavior for the main/root timeline.
