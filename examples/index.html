Document class

This is an example of a document class with the Temple. This class configures some Temple settings, connects to Yalog (for debug messages) and handles Google Analytics tracking (which is not part of the Temple of course).

You can use this class as template for your document class. This class is used as base class for all Temple examples. Since this class makes use of Google Analytics tracking, 'analytics.swc' must be added to the classpath to get it working.

FrameStableMovieClip & FrameRateKiller

This is an example of the FrameStableMovieClip and FrameRateKiller.

View this example online at: http://templelibrary.googlecode.com/svn/trunk/examples/temple/ui/animation/FrameStableMovieClipExample.swf

This example uses an .fla file which can be found at: http://templelibrary.googlecode.com/svn/trunk/examples/temple/ui/animation/FrameStableMovieClipExample.fla

This example consists of 3 animations: a normal timeline animation, a FrameStableMovieClip timeline animation and a scripted animation using TweenLite. The TweenLite animation is time based and will always animate in the same duration. The FrameStableMovieClip will also play in the same duration, no matter what the frame rate is. The frame rate is throttled using the FrameRateKiller. You can change the fps in the TextField to test it.

LiquidScrollComponentExample

This is an example of the ScrollComponent.

By setting liquid properties (left, right, top, bottom etc.) the ScrollComponent will be liquid en will automatically be resized according to the size of the stage. If you resize the example, the dimension of the ScrollComponent will be adjusted. The size of the button of the ScrollBar will also automatically be resized. The ScrollBar will be hidden when the size of the ScrollComponent is large enough to display all the content.

View this example online at: http://templelibrary.googlecode.com/svn/trunk/examples/temple/ui/scroll/LiquidScrollComponentExample.swf

This example uses an .fla file which can be found at: http://templelibrary.googlecode.com/svn/trunk/examples/temple/ui/scroll/LiquidScrollComponentExample.fla
